VR View is a name that has been used for many projects over the years. Here is a project by Axis Images that is over 20 years old and uses Apple's Quicktime.

The VR View project on this site was originally started in late 2015. It had some really cool features and I intend to eventually bring this project back here.

In the spring of 2016 Google started using the name for its product/technology used to embed immersive 360 and 3D content. The web version this project was created primarily by Boris Smus. It uses the three.js 3D library, which uses WebGL.

For now I am using this url to help out with the Google/Boris Smus version. I also have here an extra customized version which includes a server, some video player updates, and an experimental connect mode that allows the viewing of 3D 360 content with an inexpensive cell phone.
